如何通过U盘传输文件到显示屏？

将U盘中的文件通过vga转接头传到教室的电脑显示屏上。

需要VGA转接头。

请在操作前区分正反，按照步骤操作。

  1. 将投影仪的VGA接口与转接头VGA端相连。
    尤其要注意VGA的正反。
  2. 将U盘的Type-C接口插入转接头的Type-C端口。
    如果计算机屏幕显示【正在扫描U盘】，则表示连接完成。
  3. 打开计算机，在有可移动存储的设备栏下点击代表自己U盘的图标。
  4. 在打开的U盘中选择自己想要传输的文件并复制，粘贴到计算机上。
